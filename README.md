# About

Rockshelf is a Rock Band 3/Rock Band 3 Deluxe utility suite for RPCS3 users built with [Electron Vite](https://electron-vite.org/) framework. With Rockshelf, you're able to:
- Check statistics overall of Rock Band 3;
- Install Deluxe and additional patches;
- Install song packages;

# Dependencies

Rockshelf uses the [`rbtools`](https://github.com/ruggeryiury/rbtools) package, it basically needs the same Python dependencies as it does. [See the required dependencies here](https://github.com/ruggeryiury/rbtools?tab=readme-ov-file#dependencies).

You can use the Python Dependencies Installer script files to automatically check and install missing Python dependencies. You can find the script inside the `core/bin/python` directory. The scripts are translated to English, Portuguese (Brazil), and Spanish.
